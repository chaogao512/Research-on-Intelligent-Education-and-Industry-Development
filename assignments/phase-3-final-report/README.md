# Phase 3 — Final Report

## Overview

The Phase 3 assignment is the final deliverable — either a completed research report presenting empirical findings, or a comprehensive system development report if the project took a design-based approach.

| Item | Details |
|------|---------|
| **Word count** | 15,000+ words (excluding appendices and references) |
| **Format** | `.md` (draft) → `.pdf` (final submission) |
| **Due date** | — (confirm with instructor) |
| **Submission** | Tag final version as `phase3-final` in git |

---

## Objectives

1. Present a complete, publication-quality research report.
2. Demonstrate mastery of the full research cycle from problem identification to conclusion.
3. Show how the research findings connect to both theory and practice.
4. For system-development projects: document the system design, implementation, evaluation, and lessons learned.

---

## Rubric

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Research rigor | 25% | Sound methodology, appropriate analysis, valid conclusions |
| Theoretical contribution | 20% | Clear connection to and advancement of theory |
| Practical contribution | 15% | Actionable implications for practitioners |
| Writing quality | 20% | Clear, precise, academic English/Chinese |
| Evidence & citation | 10% | Appropriate evidence, comprehensive bibliography |
| Formatting & structure | 10% | Professional document structure, APA 7th ed. |
