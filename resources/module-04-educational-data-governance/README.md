# Module 04 — Educational Data Governance

> **《教育数据治理》**

---

## 🎯 Learning Objectives

By the end of this module, students should be able to:

1. Explain the principles of responsible educational data governance.
2. Identify key privacy and ethical issues arising from large-scale learning analytics.
3. Apply the PDPA, GDPR, and China's PIPL framework to educational data scenarios.
4. Evaluate data governance policies in Chinese universities.
5. Propose a data governance framework for an intelligent education system.

## 🔑 Key Concepts

- **Learning Analytics**: The measurement and analysis of learner data to optimize learning.
- **PIPL (个人信息保护法)**: China's Personal Information Protection Law, in effect since 2021.
- **Data Minimization**: Collecting only the data strictly necessary for educational purposes.
- **Algorithmic Fairness**: Ensuring AI-driven educational decisions do not systematically disadvantage learner groups.
- **Consent & Transparency**: Learner rights to understand how their data is used.
- **Data Governance Framework**: Organizational policies, procedures, and roles for managing educational data assets.
