# Module 02 — Key Technologies and Environments of Intelligent Education

> **《智能教育关键技术与环境》**

---

## 模块作用

模块 02 聚焦智能教育“技术底座 + 环境体系”，帮助学习者把握关键技术发展脉络、平台参考框架与典型应用环境，是从概念走向系统设计的重要过渡模块。

---

## 主要内容（依据模块简介）

1. 智能技术发展背景与阶段（AI、大数据、云计算）
2. 智能教育关键技术及平台参考框架
3. 智能教室、智能校园、教育云平台的结构与作用
4. 结合教育需求优化智能教育关键技术平台

> 模块简介注明：本模块包含 **4 个专题、12 节微课**。
