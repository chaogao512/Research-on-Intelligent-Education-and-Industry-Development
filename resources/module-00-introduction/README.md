# Module 00 — Introduction

> **《绪论：课程导入与学习组织》**

---

## 模块作用

模块 00 用于完成课程开场与学习准备，帮助学习者明确课程目标、学习方法、考核方式与整体节奏，是后续模块学习与小组协作的起点。

---

## 主要内容

1. 课程定位与目标（36 学时、2 学分）
2. 基于设计思维的混合式协作学习方法
3. 课程考核构成与评分规则
4. 综合作业任务说明（设计思维方法运用）
5. 11 周课程日历与模块发布节奏
