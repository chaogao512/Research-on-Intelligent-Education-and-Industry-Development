# Module 03 — Applications of Intelligent Technology in Primary and Secondary Schools

> **《中小学智能技术应用》**

---

## 模块作用

模块 03 聚焦中小学场景中的智能技术应用，强调“问题场景—技术方案—教学治理”联动分析，帮助学习者将技术认知转化为校园实践设计能力。

---

## 主要内容（依据模块简介）

1. 中小学课堂、管理与评价场景中的智能技术应用路径
2. 智能技术赋能课程教学创新与学生个性化发展
3. 校园智能化建设中的实施问题与优化方向

> 模块简介注明：本模块包含 **6 个专题、13 节微课**。
